Typeset — CSS Playground

An interactive in-browser tool for experimenting with font pairings, type scale, and CSS variable output — built to make typography decisions faster and more visual.

Year

2025

Role

Front-end Developer

Tools

HTML, CSS, JavaScript, Google Fonts API

Overview

Typeset is a small browser-based tool that lets you pick two Google Fonts (one for headings, one for body), adjust a base font size, and preview the combination on a sample page in real time. It also generates the CSS variables and @import line you'd need to copy into a project.

I built it because I kept opening multiple browser tabs to test font combinations while working on other projects, which was slow and hard to compare. Having everything in one screen made the process much faster.

Process

01

Google Fonts API

The tool uses the Google Fonts API to fetch a list of available fonts on load, then populates two dropdown menus. Selecting a font dynamically injects a link tag into the head with the correct Google Fonts URL.

02

Live Preview

The preview panel on the right updates instantly when any setting changes. It uses CSS custom properties — changing the font or scale updates a variable on the root element, and everything that references that variable updates automatically.

03

Type Scale

Added a scale ratio selector (Minor Third, Major Third, Perfect Fourth, etc.). The sizes for h1 through h6 and body text are all calculated from the base size and the ratio, and displayed both in the preview and in the generated CSS.

04

CSS Output

A code block at the bottom shows the full CSS snippet you'd need to use the current settings in a real project. Clicking it copies it to the clipboard using the Clipboard API, which I hadn't used before.

Outcome

Typeset ended up being genuinely useful — I've actually used it while working on other projects, including this portfolio. It's one of the few things I've built that I kept using after finishing it.

The most interesting technical part was working with CSS custom properties via JavaScript. Setting document.documentElement.style.setProperty() to update a variable and watching the entire page reflow in real time was a good reminder of how powerful CSS variables are when combined with a little JS.
