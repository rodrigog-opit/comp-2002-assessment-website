UX Design UI Redesign 2025

Orbitask — PM Tool Redesign

A redesign of a project management tool's task board, aimed at reducing visual clutter and making it easier to prioritise work at a glance.

Year

2025

Role

UI/UX Designer

Tools

Figma, Notion, Maze

Overview

Orbitask is a fictional project management tool that I used as the subject for a redesign exercise. The original interface had a cluttered sidebar, inconsistent use of colour to indicate priority, and a task board that felt overwhelming when a project had more than a dozen items.

The brief I set for myself was to redesign the task board view so that a user could immediately understand the state of a project — what's blocked, what's in progress, and what needs attention — without having to read every card in detail.

Process

01

Audit of the Original

Started by documenting all the usability issues in the original design — 11 in total, ranging from poor colour contrast on status badges to a sidebar with 14 items that most users ignored.

02

Competitive Analysis

Looked at how Linear, Notion, and Asana handle task boards. Linear's approach to priority indicators and Notion's collapsible sections were the most useful references.

03

Redesign

Simplified the sidebar from 14 items to 6, introduced a consistent colour-coding system for priorities, and redesigned each task card to show only the most relevant information upfront.

04

Validation

Showed both versions (original and redesign) to 3 people unfamiliar with the product and asked them to find the highest-priority blocked task. All 3 found it faster in the redesign.

Outcome

The redesign reduced the visual noise on the task board significantly. The most noticeable improvement was the sidebar — simplifying it made the board itself feel much less cramped.

The key takeaway from this project was that redesigning something existing is harder than designing from scratch — you have to decide what to keep, not just what to add. The constraint of not changing the underlying functionality forced more creative layout decisions.
